**VibeKit Studio** — Full‑stack mini‑site builder with theme generator, user authentication, CRUD, and live publishing.

---

## 🛠️ Features
- Theme generator for customizing site appearance
- User sign-up & login with authentication
- CRUD dashboard for pages & posts
- Live publishing of created sites

---

## 🧪 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js
- Deployment: Netlify
