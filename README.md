# VibeKit Studio

**VibeKit Studio** — Full‑stack mini‑site builder with theme generator, user authentication, CRUD, and live publishing.

## 🛠️ Features
- Theme generator
- User sign‑up & login
- CRUD dashboard
- Live publishing

## 🧪 Tech Stack
- Frontend: React / HTML / CSS
- Backend: Node.js
- Deployment: Netlify

## 🚀 Local Setup
1. `git clone https://github.com/svsyadav123/Full-Stack-Vibe-Coder-internship-Assignment-.git`
2. `cd your‑project‑folder`
3. `npm install`
4. `npm start`
